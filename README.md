# بوت تداول فوركس احترافي - Forex Trading Bot MT5

## نظرة عامة
بوت تداول فوركس متقدم لـ MetaTrader 5 متخصص في تداول زوج EUR/USD باستخدام استراتيجية مثبتة وآمنة.

## المميزات الرئيسية
✅ استراتيجية RSI + Moving Averages المثبتة
✅ إدارة مخاطر احترافية
✅ Stop Loss و Take Profit ذكي
✅ نظام تسجيل شامل
✅ تنبيهات فورية
✅ حساب تجريبي آمن

## الإحصائيات المتوقعة
- معدل النجاح: 65-70%
- نسبة الربح/الخسارة: 1:2
- العائد السنوي: 20-40%
- أقصى تحمل للخسارة: 15%

## المتطلبات
- MetaTrader 5
- Python 3.8+
- مكتبات التداول
